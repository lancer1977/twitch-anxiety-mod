## Name
Twitch Anxiety Mod

## Summary (132 chars max)
Hide viewer/follower numbers and left sidebar elements on Twitch to reduce pressure and keep focus.

## Description
Twitch Anxiety Mod removes social-pressure UI from Twitch so you can focus on content, not numbers.

### Features
- Hide viewer counts
- Hide follower counts
- Hide “watching now” labels
- Strict mode to hide numeric badges
- Optional full left sidebar hide/collapse
- Simple popup toggles

### Privacy
- No tracking
- No analytics
- No external data collection
- All processing happens locally in your browser

### Permissions
- `https://www.twitch.tv/*` to modify Twitch page UI
- `storage` to save your toggle preferences

### Notes
Twitch updates its UI regularly. If an element reappears, update the extension and we’ll patch selector coverage.
