# Distribution and Store Publishing Milestones

## Milestone 1: Review-ready package

- [x] Manifest version is bumped for the current release line.
- [x] ZIP artifact builds successfully.
- [x] Main install instructions cover the unpacked download path.
- [x] Store listing copy exists in the repo.
- [x] Privacy policy is aligned with local-only storage.
- [x] Screenshot checklist exists.
- [x] Release notes template exists.

## Milestone 2: Identity and verification

- [ ] Gather the ID / legal details required for Chrome Web Store publisher verification.
- [ ] Complete trader or non-trader declaration.
- [ ] Verify the publisher account.

## Milestone 3: Store submission

- [ ] Capture the screenshots listed in the checklist.
- [ ] Package the final ZIP for upload.
- [ ] Submit the extension to the Chrome Web Store.
- [ ] Record the submission date and review status.

## Milestone 4: Post-review

- [ ] Address any review feedback.
- [ ] Update docs if the store asks for new disclosure or metadata.
- [ ] Mark the roadmap complete after the item is live.
