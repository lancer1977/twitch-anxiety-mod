# Chrome Web Store Screenshots Checklist

## Required shots

- [ ] Extension popup with the main toggles visible
- [ ] Popup showing the chat-user hide list controls
- [ ] Twitch page with hidden viewer or follower counts applied
- [ ] Twitch chat page with listed users hidden

## Capture notes

- Use a normal Twitch page state that makes the feature obvious.
- Keep the browser zoom and popup size consistent across screenshots.
- Avoid showing private or personal chat content.
- Make sure the feature is actually enabled in the shot.

## Upload notes

- [ ] Confirm image dimensions meet Chrome Web Store guidance.
- [ ] Keep filenames short and descriptive.
- [ ] Retake any screenshot that is blurry or ambiguous.
